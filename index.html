Notions de base des VARIABLES

Il y a plusieurs types de variables: LET, VAR et CONST.
LET est une variable qui va changer de valeur, par exemple elle compte les points de la page.
CONST est une variable qui ne doit pas changer de valeur. Par exemple une balise.
VAR est amenée à disparaitre.

J'annonce d'abord la variable et ensuite je lui donne l'action à réaliser. Pour cela je cible mon élément dans mon html avec un ID.



Puis dans mon JS, je cible mon html avec:


document.getElementById('variable');



Puis je déclare ma variable et je lui donne son action.


J'annonce :

const.variable = document.ElementById('variable');


variable.style.color = green;

Enfin, j'ajoute:


console.log(variable);


ainsi je vérifie ma console.



Si je souhaite donner une nouvelle valeur (par exemple : 2) à ma variable nommée "variable"
alors je l'écris sans annoncer que c'est une variable puisqu'elle a déjà été déclarée.
Au lieu de redéclarer CONST.VARIABLE je vais écrire:


variable = 2;



Puis je vérifie avec :

console.log(variable);
